Class I: Soil has little to no limitations that restrict its use.

Class II: Soil requires moderate conservation practices.

Class III: Soil requires significant management intervention.

Class IV: Soil has very severe limitations and requires very careful management.

Class V: Soil is subject to little or no erosion but has other limitations which may make it unsuitable for crop cultivation.

Class VI: Soil has severe limitations that make it unsuitable for cultivation but may be used for pasture, rangeland etc.

Class VII: Soil has very severe limitations that make it unsuitable for cultivation but may be used for pasture, rangeland etc.

Class VIII: Soil has very severe limitations that preclude crop production and restricts it's use to recreational purposes, pasture, rangeland etc.

Class 2: 1999-2166mm

Class 3: 2167-2310mm

Class 4: 2311-2478mm

Class 5: 2479-2700mm

All Rainfall Amounts are in Millimetres per year.
