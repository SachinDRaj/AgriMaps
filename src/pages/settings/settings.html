Fetch Radius

The application at default fetches all data in a range of 1000 meteres from location. To change this please tap the settings button in bar and set the radius fetch to desired number.

Profile Mode / Recommender Mode

To switch the application between the default profile mode and the recommender mode, tap the R or P button in the action bar. After tapping this button a notice will appear at the bottom of your screen and the views in the left hand drawer will change.

Location

To change the location of interest, press screen at desired point for a minimum of 5 seconds. A prompt will appear to confirm that you would like to change the interest location to that point.
