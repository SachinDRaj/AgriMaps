Fetch Radius

This application, at default fetches all data in a range of 1000 metres from location specified. To change this please tap the app button on the top navigation bar and set the radius to desired number and choose the catergory you would like the data to be generated for.

Profile Mode / Recommender Mode

To switch the application between the default profile mode and the recommender mode, tap the R or P button on the bottom tab bar. After tapping this button a notification will appear at the middle of your screen notifying you of which mode you are in.

Location

To change the location of interest, double click the screen at any desired point. A prompt will appear to confirm that you would like to change the point of interest to the location specified.
